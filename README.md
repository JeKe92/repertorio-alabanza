# Separated Index Project

This project demonstrates how to separate the contents of an HTML file into distinct JavaScript, CSS, and HTML files for better organization and maintainability.

## Project Structure

```
separated-index-project
├── src
│   ├── index.html        # Main HTML structure of the web application
│   ├── css
│   │   └── styles.css    # CSS styles for the application
│   └── js
│       └── main.js       # JavaScript code for dynamic behavior
├── .gitignore            # Files and directories to ignore in version control
├── package.json          # npm configuration file
└── README.md             # Documentation for the project
```

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd separated-index-project
   ```

3. Install dependencies (if any):
   ```
   npm install
   ```

4. Open `src/index.html` in a web browser to view the application.

## Usage

- The `index.html` file serves as the entry point for the application.
- The `styles.css` file contains all the styling rules for the HTML elements.
- The `main.js` file includes the JavaScript code that adds interactivity to the web page.
